Pour que le projet fonctionne :

	Mettre l'adresse IP, le login et le mot de passe de sa base de donnée dans le fichier :
		- configbdd.php

	Dans phpMyAdmin :
		- importer le fichier "netto1.sql" se trouvant dans le dossier Document

Il ne reste plus qu'à lancer le projet en configurant le fichier host de windows et vhost de XAMPP ou WAMP (les logins/mot de passes se trouvent dans la table user de la base)
